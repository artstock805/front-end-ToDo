# 일일 브리핑 스킬

하루 브리핑을 생성합니다. 구성: **① 날씨 → ② 환율·경제(+알림) → ③ 주요 뉴스 → ④ 오늘 할 일**.
사용자가 카카오톡 전송을 요청하면 마지막에 요약을 카톡으로 보냅니다.

## ⚙️ 설정 (사용자가 원하면 이 값만 바꾸면 됨)
- **지역(날씨)**: `서울`
- **환율 알림 기준(원/달러)**:
  - `1,450원 이상` → ⚠️ 주의
  - `1,500원 이상` → 🚨 경고
  - 전일 대비 `±10원 이상 급변` → 📈 급등 / 📉 급락 표시
- **카카오 전송**: 사용자가 "카톡으로 보내줘"라고 할 때만. 기본은 채팅 표시만.

## 기준 날짜
- 세션 컨텍스트의 `currentDate`를 "오늘"로 삼습니다. **"전날"은 오늘 −1일**입니다.
- 뉴스는 "전날 발생/보도된" 것 중심, 필요하면 오늘 이른 시각까지 포함.
- 검색 질의에 실제 날짜(YYYY-MM-DD)나 "어제/오늘" 같은 최신성 키워드를 넣어 최신 결과를 얻으세요.

## 수집 절차 (WebSearch / WebFetch 사용)

### 1) 날씨
- 설정된 지역의 **오늘 날씨**를 검색합니다 (예: "서울 오늘 날씨").
- 최고/최저 기온, 강수확률/강수여부, 특보(폭염·한파·미세먼지 등)를 한 줄로.
- 우산·겉옷 등 **실용 팁 한 마디**를 덧붙입니다 (예: "오후 비 → 우산 챙기세요").

### 2) 환율·경제 (뉴스보다 우선)
- **환율**: 원/달러(USD/KRW) 필수. 필요 시 원/엔(JPY), 원/유로(EUR), 달러인덱스도.
  최신 종가/현재값과 **전일 대비 등락(▲/▼, 원·%)**을 함께.
- **환율 알림**: 위 설정 기준에 걸리면 해당 이모지(⚠️/🚨/📈/📉)와 함께 굵게 강조하고,
  브리핑 맨 위 또는 환율 줄 옆에 눈에 띄게 표시합니다. 기준 이하이면 표시하지 않습니다.
- **경제 지표·뉴스**: 코스피/코스닥, 미국 증시(다우·나스닥·S&P500), 국제유가, 금리·물가.
  한국은행·연준 등 정책 이슈가 있으면 우선.
- 서로 다른 검색 질의 2~4개를 병렬로 던지세요
  (예: "USD KRW 환율 오늘", "코스피 마감 어제", "미국 증시 마감", "국제유가").

### 3) 주요 헤드라인 뉴스 (중요도 순)
- **해외**와 **국내**를 나눠서 검색.
- 중요도: 파급력(경제·사회·안보 영향) > 화제성 > 지속성. 여러 매체가 크게 다룬 사안일수록 상위.
- 각 항목은 **한 줄 요지 + (출처 링크)**. 해외 3~5개 / 국내 3~5개.
- **경제·금융 헤드라인은 다른 주제보다 위에** 배치(사용자 우선순위).

### 4) 오늘 할 일 (Todo 통합)
- `mcp__todo-app__list_todos`를 `today: true`로 호출.
- 도구 미연결/실패 시 이 섹션은 조용히 생략하고 나머지는 정상 진행.

## 출력 형식 (채팅용 — 전체)

```
# 📅 {오늘 날짜} 일일 브리핑

## 🌤️ 날씨 — {지역}
- {날씨 요약, 최고/최저} · 💡 {실용 팁}

## 💱 환율 · 경제   {알림 이모지 있으면 여기에}
- 원/달러: 1,4xx.x원 (전일 대비 ▲/▼ x.x)  {⚠️/🚨/📈/📉}
- 코스피 ... / 미국 증시 ... / 유가 ...
- (경제 핵심 뉴스 1~3줄)

## 🌍 주요 뉴스
**해외**
1. (요지) — (출처)
**국내**
1. (요지) — (출처)

## ✅ 오늘 할 일
- (list_todos today 결과)
```

## 카카오톡 전송 (요청 시)
카카오 기본 텍스트 메시지는 **200자 제한**이 있으므로, 전체 브리핑이 아니라 **핵심 요약**을 만듭니다.

1. 아래 형식의 **200자 이내** 요약 텍스트를 작성:
   ```
   📅 {날짜} 브리핑
   🌤️ {지역} {날씨 한줄}
   💱 원/달러 {값}({등락}) {알림이모지}
   📰 해외 {n}·국내 {n}건 / 핵심: {가장 중요한 뉴스 1건}
   ✅ 오늘 할일 {n}건
   ```
2. `mcp__todo-app__send_kakao` 도구를 `text`={요약}, `link`="http://localhost:3456" 로 호출.
3. 성공하면 "카톡으로 보냈어요"라고 알리고, 전체 브리핑은 채팅에도 그대로 표시합니다.
4. 도구가 실패하면(설정 미완료 등) 실패 사유를 전하고, 카카오 설정 방법(README의 "카카오톡 설정")을 안내합니다.

## 품질 기준
- **최신성 검증**: 결과 날짜 확인, 오래된 기사 제외. 환율·지수는 최신값.
- **출처 명시**: 뉴스마다 매체명·링크. 수치는 근거 있는 값만.
- **균형**: 사실 위주, 정파 치우침 없이. 확인 안 된 속보는 "미확인" 표시.
- **간결**: 항목당 1~2줄. 30초 안에 훑을 수 있게.
- 원하면 결과를 보기 좋은 HTML 아티팩트로도 만들 수 있음(사용자 요청 시).
